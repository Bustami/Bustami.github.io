Historical Men FIFA Ranking [Apr-2020]

Apr 25, 2020 #fifa-ranking #rstats #shiny #football #soccer

I just updated the R Shiny app which allows you to visualize the historical Men FIFA ranking interactively.

  • Data (from August 1993 to April 2020) scraped from the FIFA official website.

  • The app has a date range and country selectors. It is possible to visualize from 1 to 3 nations at the same time.

  • Option to image download as PNG file.

  • Link to the Github repo if you want to review the code and/or use the data by yourself.
