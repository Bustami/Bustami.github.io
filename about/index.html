Acerca de

DATOFUTBOL nace como un proyecto que busca aportar al debate y conocimiento de todos los aspectos futbolísticos que puedan analizarse de manera objetiva utilizando datos.

Para la construcción del sitio web usamos la librería blogdown de R, la cual permite utilizar el generador de sitios estáticos Hugo en conjunto con el lenguaje de marcado para documentos RMarkdown. También, en algunos post incrusto aplicaciones interactivas creadas con Shiny.

Cuando es posible, comparto abiertamente en Github la base de datos y los códigos utilizados, con el objetivo de que los análisis sean reproducibles y también para aportar a la consigna Open Data.
